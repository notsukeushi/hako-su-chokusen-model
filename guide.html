なぜ箱数直線モデルを学ぶのか（日本語）

このアプリは、小学3年生が数直線の考え方を、無理なく、しかししっかり身につけるための練習です。計算ドリルだけでは育ちにくい「構造の理解」を、箱と点の絵で支えます。

どんな目的に役立つか

  • 数直線の理解 … 左・右の点、きょり（差）、並べ方を一本の線で考える
  • たし算・ひき算の意味 … 「右 − 左」「左 ＋ 箱の数」が、図の上の動きとしてイメージできる
  • 数え方の正確さ … 箱を数えるとき、名前の数字だけでなく点の位置で数える必要があると学ぶ（はじめと終わりを含む、など）
  • 記号の入口 … 箱の名前（3:）やあだ名（1番・みどり）は、のちの文字式・座標の「別の呼び方」に近い
  • 中学以降の土台 … 数直線上の区間、座標、一次関数のグラフなどにつながる考え方の入り口になる

何が良いのか（このモデルの強み）

  • ひとつの絵でいろいろな問題が解ける（きょり・箱の数・名前・あだ名）
  • 見た目で考えられるので、「なぜ 8−3 なのか」が説明しやすい
  • よくある間違い（7:−3:＝4こ、など）に気づき、直し方がはっきりする
  • 段階的 … パート1→2→3と、むずかしさが少しずつ増える
  • 何度でも練習できる（乱数問題）ので、型を理解したあと定着しやすい
保護者の方へ：正解を覚えるより、「図のどこを見て、どんな式を立てたか」を一緒に話すと、効果が大きいです。

Why learn the Box Number Line Model? (English)

This free app helps Grade 3 students build a clear picture of the number line—not only drill calculations. Boxes and points give a visible structure that word problems alone often hide.

What goals it supports

  • Number line sense … left/right points, distance as a difference, arranging objects in order
  • Meaning of + and − … “right − left” and “left + number of boxes” match moves on the line
  • Careful counting … learn to count boxes by positions (points), not only by labels on boxes
  • Toward symbols … names like 3: and nicknames like “1st” or “Green” preview variables and coordinates
  • Foundation for later math … intervals on a line, coordinates, and graphs in middle school

What is good about this model

  • One picture, many question types … distance, number of boxes, real names, nicknames
  • Visual reasoning … easier to explain why we use 8 − 3, not just memorize
  • Common mistakes become clear … e.g. why 7: − 3: is not the number of boxes
  • Step-by-step parts … difficulty increases gently across three parts
  • Repeatable practice … new numbers each time you press “New problems”
For parents: ask “What did you look at on the diagram?” more than “What is the answer?”

箱数直線モデルとは？

すうちょくせん（数直線）の上に、大きさが 1 の箱を横に並べて考える方法です。次の3つを、同じ絵の中で結びつけて学びます。

  • 点の名前 … 数直線の目盛り（左はじ・右はじなど）
  • きょり … 2つの点の間の長さ
  • 箱の数 … 並べた箱が何こあるか

この問題集では、左はじの点の名前は 1〜20、右はじの点の名前は 30〜49 の範囲で出題します（計算しやすい数字にしています）。

① 点の名前と「きょり」

数直線には、左と右に 点 があります。その点についている数字を「点の名前」と呼びます。

左はじの点の名前 3 右はじの点の名前 8 |---------------------------| 3 8

2つの点のあいだの長さを きょり といいます。

きょり ＝ 右はじの点の名前 − 左はじの点の名前

例：3 から 8 まで → 8 − 3 ＝ 5

ポイント：「右の数 − 左の数」で、点と点の間の長さが求まります。

② 箱の数と「きょり」は同じ

大きさが 1 の箱を、すき間なく並べると、箱の数 と きょり は同じ数字になります。

[3:] [4:] [5:] [6:] [7:] |----|----|----|----|----| 3 8 ← 箱が5こ → きょりも5 →
箱の数 ＝ きょり（箱の大きさが1のとき）

だから、左はじの点の名前と箱の数がわかれば、「左 ＋ 箱の数」で右はじの点の名前が求まります。逆に、右はじと箱の数がわかれば、「右 − 箱の数」で左はじが求まります。

③ 箱の「名前」

それぞれの箱には 名前 があります。ルールは次のとおりです。

  • 箱の左はじが、どの点の上にあるかで決まる
  • 数字のうしろに ：（コロン）をつける → 例：3:
点 3 の上に箱 → 名前は 3: 点 7 の上に箱 → 名前は 7:
「3:」は「左端が 3 の箱」という意味です。点の名前「3」と、箱の名前「3:」は、数字の部分が同じです。

④ 箱が何こあるか数える（重要）

箱 3: から 箱 7: まで（7: もふくむ）何こ？ という問題では、箱の名前同士を引くだけ では足りないことがあります。

箱の数は、点 同士の差で数えます。

箱の数 ＝（右の点の名前）−（左の点の名前）

例：点 3 から点 8 まで箱を並べる → 8 − 3 ＝ 5こ（箱 3: 〜 7:）

別の言い方：（いちばん右の箱の右はじの点）−（いちばん左の箱の左はじの点）

箱 a: 〜 b:（b: もふくむ）の箱の数 ＝ (b ＋ 1) − a
7: − 3: ＝ 4 ではなく、点 8 − 点 3 ＝ 5 が正解、という考え方です（問15で学びます）。

⑤ あだ名（エイリアス）

箱には、本名のほかに あだ名 をつけることができます。

  • 「1番」「2番」… 左から数えた順番
  • 「みどり」「あか」… 好きな名前
  • 「14:」のように数字＋：のあだ名 … 本名との差が一定（＋10 など）

あだ名が変わっても、箱が並んでいる位置（本名） は変わりません。あだ名から本名を求める問題では、「あだ名同士の差 ＝ 本名同士の差」と考えるとよいです。

⑥ 問題集の3パート

  • パート1 … 点の名前・きょり・箱の数（箱の名前はまだ出ない）
  • パート2 … 箱の名前（3: など）を使う
  • パート3 … あだ名（1番、みどり など）を使う

図は、箱がたくさんあっても 左2こ・・・・・いちばん右1こ だけ描きます。・・・・ のあいだにも箱が続いている、と考えてください。

⑦ おうちでの使い方（保護者の方へ）

  • まずパート1で「きょり ＝ 右 − 左」を確かめる
  • パート2で「箱の名前 ＝ 左端の数字：」に慣れる
  • パート3は、本名とあだ名の対応を一緒に図で確かめる
  • 「新しい問題」で数字が変わるので、同じ型を何度も練習できます

What is the Box Number Line Model?

It is a way to learn on a number line by placing boxes of size 1 next to each other. You connect three ideas in one picture:

  • Point names … numbers on the ticks (left end, right end, etc.)
  • Distance … the length between two points
  • Number of boxes … how many boxes are in a row

In this workbook, the left point name is between 1 and 20, and the right point name is between 30 and 49 (easier numbers for Grade 3).

① Point names and distance

The number line has a left end and a right end. The number at each tick is called the point name.

Left point name 3 Right point name 8 |---------------------------| 3 8

The length between two points is the distance.

distance = right point name − left point name

Example: from 3 to 8 → 8 − 3 = 5

Tip: “right number minus left number” gives the distance.

② Number of boxes equals distance

When each box has size 1 and they touch with no gaps, the number of boxes equals the distance.

[3:] [4:] [5:] [6:] [7:] |----|----|----|----|----| 3 8 ← 5 boxes → distance is also 5 →
number of boxes = distance (when box size is 1)

So if you know the left point name and the number of boxes, the right point name is: left + number of boxes. The other way: right − number of boxes = left.

③ Box names

Each box has a name:

  • It is decided by the left edge of the box on the number line
  • Write the number plus a colon → e.g. 3:
Box starts at point 3 → name is 3: Box starts at point 7 → name is 7:
“3:” means “the box whose left edge is at 3.” The digit matches the point name at that place.

④ Counting boxes (important)

For “from box 3: to box 7: (including 7:), how many boxes?”, subtracting box names alone is not enough.

Count boxes using points:

number of boxes = (right point name) − (left point name)

Example: boxes from point 3 to point 8 → 8 − 3 = 5 boxes (boxes 3: through 7:)

boxes from a: through b: (include b:) = (b + 1) − a
7: − 3: = 4 is wrong; point 8 − point 3 = 5 is correct (see Part 2, Question 15).

⑤ Aliases (nicknames)

Boxes can have nicknames as well as real names:

  • 1st, 2nd, 3rd … order from the left
  • Green, Red … any words
  • Number nicknames like 14: … a fixed gap from the real name (+10, etc.)

The nickname can change, but the box stays in the same place. To find real names, use: gap between nicknames = gap between real names.

⑥ Three parts in this workbook

  • Part 1 … point names, distance, number of boxes
  • Part 2 … box names (3:, etc.)
  • Part 3 … aliases (1st, Green, etc.)

Diagrams show only 2 boxes on the left ···· 1 box on the right. Imagine boxes continuing in the “····” part.

⑦ For parents at home

  • Start with Part 1: “distance = right − left”
  • Part 2: get used to “box name = number at left edge + colon”
  • Part 3: match nicknames to real names on the diagram together
  • Press “New problems” to practice the same types with new numbers
